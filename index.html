First-time Setup
Connect to GitHub
Cards are stored in a private GitHub Gist so they're accessible from any device. You only need to do this once.
1
Go to github.com/settings/tokens/new and create a token. Under Scopes, check only gist. Set expiry to No expiration.
2
Copy the token (starts with ghp_) and paste it below. It is stored only in this browser and never sent anywhere except GitHub.
3
Click Connect. A private Gist will be created automatically to store your cards database.
Your token is saved in this browser's localStorage and used only to read/write your private Gist. Anthropic and Dumore's servers never see it.
